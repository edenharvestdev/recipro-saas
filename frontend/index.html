รวมขั้นตอนทั้งหมดของระบบ สำหรับน้องหน้าร้านและคนทำสูตร — กดหัวข้อเพื่อดูรายละเอียด

ลำดับการทำงานของระบบ
วัตถุดิบ (ซื้อเข้า/ตั้งหมวด) → สูตร (ส่วนผสม + ต้นทุน) → นำขึ้นเมนู + ตัวเลือก → ขายหน้า POS → ระบบตัดสต๊อกวัตถุดิบให้อัตโนมัติ
1. วัตถุดิบ & หมวดสินค้า
เพิ่มวัตถุดิบ: หน้า "วัตถุดิบ" → กด "เพิ่มวัตถุดิบ" → ใส่ชื่อ ราคา/หน่วย คงเหลือ จุดเตือนใกล้หมด แล้วเลือก หมวดสินค้า
หมวดที่ใช้บ่อย:
• RAW (วัตถุดิบ) — ของที่ใช้ในสูตร เช่น นม ผงชา ไซรัป → ตัดสต๊อกตอนขายผ่านสูตร
• SALE (ของขาย) — ขายตรงไม่ผ่านสูตร เช่น ขนมซื้อมาขายต่อ น้ำขวด → ตัดสต๊อกเองตอนขาย + คิดกำไร
• PACKAGING (แพ็กเกจ) — แก้ว ฝา หลอด
• ASSET (ทรัพย์สิน) — เครื่องมือ ไม่ตัดสต๊อกตอนขาย
ใส่หมวดให้ถูก ระบบถึงจะตัดสต๊อก/คิดต้นทุนถูก
2. ทำสูตร & ของกลาง (ครัวทำเอง)
สูตรเมนู: หน้า "สูตร/เมนู" → "สร้างสูตร" → ใส่ชื่อ + ส่วนผสม (เลือกวัตถุดิบ + จำนวนต่อแก้ว) → ระบบคิดต้นทุนให้อัตโนมัติ → เปิด "นำขึ้นเมนู" เพื่อให้ขายได้
ของกลาง (เช่น Topping Cream Cheese, ซอสที่ครัวทำเอง):
1. สร้างเป็น "สูตร" ของมันเอง (แต่ ไม่ต้อง นำขึ้นเมนู)
2. ผลิตเก็บสต๊อกที่หน้า "สต๊อก/ผลิต" → กด "ผลิต" (ปุ่มลัดข้างรายการ)
3. เอาของกลางนี้ไปใส่เป็น "ส่วนผสม" ในสูตรเมนูอื่นได้เลย → เวลาขายจะตัดจากสต๊อกของกลาง
3. ตัวเลือก (หวาน/ขนาด/เปลี่ยนนม/ท็อปปิ้ง)
ตั้งที่หน้า "ตัวเลือก" เท่านั้น (ในฟอร์มสูตรมีปุ่มลัด "จัดการ/เพิ่มตัวเลือก →")
กด "สร้างกลุ่มตัวเลือก" → ตั้งชื่อกลุ่ม → "เพิ่มตัวเลือก" → เลือก effect แล้ว "ใช้กับเมนู" (ติ๊กสูตรที่ใช้):
• ข้อมูลเท่านั้น — แค่บอก เช่น ร้อน/เย็น (ฟรี ไม่ตัดสต๊อก)
• ADD — เพิ่มของ + ตัดสต๊อก เช่น เพิ่มไข่มุก ช็อตเอสเพรสโซ
• REPLACE — เปลี่ยนวัตถุดิบ เช่น นมสด→นมโอ๊ต
   ① เลือก "วัตถุดิบเดิมที่จะเอาออก" → ② กด "+ เพิ่มวัตถุดิบ" เลือกตัวใหม่ + จำนวน
• QUANTITY — กำหนดปริมาณ เช่น หวาน 0/5/8g → เลือกวัตถุดิบ (น้ำตาล) + ใส่จำนวน (0 = ไม่ใส่)
• RECIPE_VARIANT — สลับทั้งสูตร เช่น ร้อน/เย็น ที่ใช้สูตร+แก้วต่างกัน
ช่อง "role (ขั้นสูง)" ข้ามได้เลย — ใช้เฉพาะเวลาอยากใช้ตัวเลือกเดียวกับหลายสูตรพร้อมกัน · "+ราคา" = ค่าเพิ่มเฉพาะตัวเลือกนั้น
4. ขายหน้าร้าน (POS)
หน้า "ขาย & ออกบิล" → กดเมนู → เลือกตัวเลือก (ถ้ามี) → ใส่ตะกร้า → รับเงิน/ออกบิล
ระบบจะ ตัดสต๊อกวัตถุดิบ ตามสูตร + ตัวเลือกให้อัตโนมัติ
คำเตือนวัตถุดิบ: ถ้าวัตถุดิบในเมนูใกล้หมด/หมด จะขึ้นเตือนที่ปุ่มเมนู
ต้องเปิด "ทำสดตอนสั่ง" ในหน้าตั้งค่าก่อน ระบบถึงจะอ่านวัตถุดิบ+ตัดสต๊อกตอนขาย (ถ้าปิด เมนูจะขึ้น "หมดสต๊อก")
5. ตัดของเสีย
หน้า "สต๊อก" → ที่รายการวัตถุดิบ เลือก "ตัดของเสีย" → ใส่จำนวน + เหตุผล (ทำหก/ทำผิดเมนู/หมดอายุ) + ชื่อคนทำ
ระบบจะหักออกจากสต๊อกและเก็บประวัติไว้ (ดูได้ในประวัติการเคลื่อนไหว)
6. สต๊อก · รับของเข้า · แจ้งเตือนใกล้หมด
รับของเข้า: หน้า "สต๊อก" → ใส่จำนวนที่ซื้อเข้าในช่อง "รับเข้า" → ยอดคงเหลือเพิ่มให้อัตโนมัติ
แจ้งเตือนใกล้หมด: ตั้ง "จุดเตือน" ที่วัตถุดิบแต่ละตัว → พอต่ำกว่าจะขึ้นเตือนให้สั่งซื้อ (รวมของขาย SALE ด้วย)
ของกลาง: กด "ผลิต" เพื่อเติมสต๊อกของที่ครัวทำเอง
7. รายรับ-รายจ่าย & เงินสดย่อย
หน้า "รายจ่าย" → บันทึกค่าใช้จ่าย (ค่าวัตถุดิบ ค่าเช่า ค่าจ้าง ฯลฯ) เลือกหมวด + แนบสลิปได้
ตั้ง "รายจ่ายประจำ" (ค่าเช่า/เน็ต) ให้เตือนทุกเดือน · จัดการ "เงินสดย่อย" (petty cash) ได้
ยอดขายจาก POS เข้ารายงานอัตโนมัติ → ดูกำไร/ขาดทุนได้ที่หน้า "รายงาน"
8. ข้อควรระวัง (อ่านก่อนใช้!)
1. เปิดแท็บเดียว — อย่าเปิดระบบหลายแท็บ/หลายเครื่องพร้อมกันแล้วแก้ข้อมูล จะเขียนทับกัน ข้อมูลหาย
2. หลังมีอัปเดตระบบ กด Ctrl + F5 ทุกเครื่อง เพื่อโหลดเวอร์ชันใหม่
3. เปิด "ทำสดตอนสั่ง" ในหน้าตั้งค่า ระบบถึงจะตัดสต๊อกวัตถุดิบตอนขาย
4. ใส่หมวดสินค้าให้ถูก — เป็นหัวใจของการตัดสต๊อก+คิดต้นทุน
